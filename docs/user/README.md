# Wombat User Guide

Wombat is a cross-platform tool for authoring and editing **funscripts** — the
JSON files that drive haptic devices in sync with a video. It is designed to feel
familiar if you have used OpenFunscripter (OFS), while adding a non-destructive
**layer** system on top of every channel.

## What Wombat is not

Wombat does not generate scripts from video with AI — that is explicitly out of
scope. It is an editor: bring a draft (your own, or from another tool) and refine
it, or build one from scratch.
